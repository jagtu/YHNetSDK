# YHNetSDK


## Installation

ios 通用网络框架, YHNetSDK is available through [CocoaPods](https://cocoapods.org). 

### 引入
To install it, simply add the following line to your Podfile:

```ruby
pod 'YHNetSDK'
```

### 打包发布方式

```ruby
pod package YHNetSDK.podspec --force --no-mangle --exclude-deps
```
